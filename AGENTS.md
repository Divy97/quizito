# AGENTS.md

Instructions for AI coding agents working in this repo.

## Agent skills

### Issue tracker

Issues live as markdown files under `.scratch/<feature-slug>/` in this repo (local-markdown tracker, no remote issue system). See `docs/agents/issue-tracker.md`.

### Triage labels

Default 5-role vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout: one `CONTEXT.md` and `docs/adr/` at the repo root cover the whole product. See `docs/agents/domain.md`.
